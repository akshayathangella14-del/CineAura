# CineAura UI Design System

## Design Philosophy

CineAura should feel:

* Cinematic
* Premium
* Intelligent
* Futuristic
* Elegant

Avoid:

* Clutter
* Bright colors
* Cheap gradients
* Excessive animations

# Theme Rules

Always Dark Theme.

No Light Mode.

No Theme Switching.

The visual identity must remain consistent.

---

# Typography

Primary Font:

Inter

Fallback:

sans-serif

---

40px

28px

---

# Card Design

Cards must have:

* rounded corners
* subtle shadows
* smooth hover animation

Hover:

* slight scale
* slight glow
* image zoom

No aggressive movement.

---

# Animation Rules

Library:

Framer Motion

Allowed:

* fade
* slide
* scale
* blur reveal

Avoid:

* bounce
* spin
* flashy effects

Animation should feel cinematic.

---

# Login Page Design

Layout:

Left:
Trending movie backdrop

Right:
Glassmorphism login panel

Display:

* CineAura Logo
* Tagline
* Login
* Register
* Google Login

Trending movie should come from backend.

---

# Home Page Design

Structure:

1 Hero Section
2 Perfect Picks
3 Because You Watched
4 Similar User Picks
5 Trending
6 Journey
7 Aura

---

# Movie Details Page

Must contain:

* Backdrop
* Poster
* Rating
* Genres
* Trailer
* Cast
* Reviews
* Recommendation Explanation

---

# Search Experience

Google-like search.

Features:

* suggestions
* fuzzy matching
* keyboard navigation
* instant results

---

# Avatar Design

Avatars must:

* be circular
* support glow ring
* support hover effect

---

# Admin Design

Admin theme must remain consistent with CineAura theme.

No separate admin theme.

---

# Visual Goal

The interface should feel like:

A premium streaming platform combined with an intelligent AI assistant.
